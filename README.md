# 🎉 Event Management System

🚀 Developed a **Full-Stack Event Management System** using **React.js (Vite), Node.js, Express.js, MongoDB (Mongoose), and Custom CSS**.

📂 Implemented a clean and scalable **MVC (Model-View-Controller) Architecture** for better project organization and maintainability.

🎨 Designed a modern, responsive, and user-friendly interface with a consistent UI/UX across all pages.

📱 Built a fully responsive layout optimized for desktop, tablet, and mobile devices.

➕ Implemented complete **CRUD Operations** to Create, Read, Update, and Delete event records.

📋 Added an intuitive event form to manage details such as Event Name, Description, Category, Venue, Date, Time, Organizer, and Event Banner.

🖼️ Displayed events using attractive and responsive event cards with images and essential event information.

🔍 Integrated a search feature to quickly find events by their name.

📝 Developed a dedicated Event Details page for viewing complete event information.

✏️ Enabled seamless editing of existing events through pre-filled update forms.

🗑️ Added secure delete functionality with a confirmation dialog before removing events.

🌐 Built RESTful APIs using Express.js for efficient frontend-backend communication.

🗄️ Integrated MongoDB with Mongoose for reliable event data storage and management.

⚡ Implemented client-side validation and loading indicators to improve user experience.

🧩 Built reusable React components to keep the frontend modular and maintainable.

📁 Organized the project into a clean folder structure for better scalability and readability.

💡 A beginner-friendly full-stack project demonstrating React.js, Express.js, MongoDB, REST APIs, responsive web design, and complete CRUD functionality.
